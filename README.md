# Collab Project 2: Pet Boarding Web Application 
> This is a simple site that lets users create accounts and login to create reservations for their pet that needs boarding.


## Technologies Used

* JavaScript
* jQuery
* MySQL
* Node
* Express
* Handlebars
* Passport.js
* Bootstrap 4
* HTML
* CSS
